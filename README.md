## Development

### Grab development dependencies
```
pip3 install -r requirements.txt
```

Assets require [yarn](https://yarnpkg.com/lang/en/docs/install/)

### Grab UI dependencies
```
yarn install
npm install -g gulp
gulp
```

### Used Packages
ORM: [flask_sqlalchemy](http://flask-sqlalchemy.pocoo.org)
